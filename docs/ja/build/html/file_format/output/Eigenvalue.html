4.3.22. Eigenvalue.dat¶

(FullDiagでのみ出力)全対角法で計算したエネルギーを出力します。以下にファイル例を記載します。
0 -4.8141698096
1 -3.7968502453
2 -3.2462822372
…
397 13.9898305290
398 14.4896221034
399 14.8525199079

ファイル形式¶

以下のファイル形式で記載されます。

  • \([\)int01\(]\) \([\)double01\(]\)

パラメータ¶

  • \([\)int01\(]\)

    形式 : int型

    説明 : 固有値のindex。基底エネルギーを0として、エネルギーの低い順から採番する。

  • \([\)double01\(]\)

    形式 : double型

    説明 : エネルギー\(\langle H \rangle\)。
